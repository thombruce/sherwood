# Sherwood

## Welcome to Your Sherwood Site

This is your homepage. Edit `content/index.md` to customize your content.

## Getting Started

1. **Start development server**:
   ```bash
   sherwood dev
   ```

2. **Edit this file** to update your homepage content

3. **Generate your static site**:
   ```bash
   sherwood generate
   ```
