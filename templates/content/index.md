# Hello, Sherwood!

Welcome to your new Sherwood site! This is a static site generator written in Rust.

## Getting Started

1. Edit this file in `content/index.md` to customize your homepage
2. Add more Markdown files to the `content/` directory
3. Run `sherwood dev` to start the development server
4. Run `sherwood generate` to build your static site
