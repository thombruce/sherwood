# Guide

Sherwood is a library first and a binary second. When the bundled template
isn't enough, add `sherwood` as a dependency, define your own templates, and
drive the build yourself.

- [Using the library](/guide/library-api/) — build a site with your own
  templates and a render closure.
- [Custom parsers](/guide/custom-parsers/) — teach Sherwood new content formats.

The page you're reading, and the rest of this site, is built exactly this way —
look at `site/` in the repository for the whole thing.
